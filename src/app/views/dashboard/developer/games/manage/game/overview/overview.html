Hide Game

Your game page is currently published and active. You can hide it to remove it from the game listings. People with the link will still be able to view it.

Cancel Game

You can mark your game as a canceled game. This will signal to everyone that you're no longer working on it. People will still be able to view its game page, but it will no longer show in the public game listings.

You should only use this if you stopped developing the game completely. If you transitioned development off of Game Jolt, you must remove your game page instead.

Remove Game

Removing your game page will remove it from the site completely. This is permanent!

You can't remove games with active sales at this time.
